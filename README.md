# AI데이터융합학과 상담 시스템

학과 재학생과 관리자(교수·조교)가 1:1로 실시간 상담을 주고받을 수 있는 웹 서비스입니다.

🔗 **[상담 시스템 바로가기](https://wonje-tech.github.io/inquiry_page/)**

---

## 이용 방법

### 학생

1. 위 링크에 접속합니다.
2. **학번**과 **이름**을 입력하고 로그인합니다.
3. 질문을 입력하면 관리자에게 전달됩니다.
4. 관리자가 답변하면 채팅창에 표시됩니다.

> 재학생 명단에 등록된 학번·이름만 로그인이 가능합니다.  
> 로그인이 되지 않는 경우 학과 사무실로 문의하세요.

---

## 주요 기능

- 학번·이름 기반 본인 인증 (원본 명단은 서버에 저장되지 않음)
- 관리자와 학생 간 실시간 1:1 채팅
- 매일 자정 채팅 자동 초기화

---

## 개인정보 안내

- 로그인 시 입력한 학번·이름은 서버에 저장되지 않습니다.
- 채팅 내용은 당일에만 유지되며, 매일 자정에 자동으로 삭제됩니다.
- 본 서비스는 학과 내부 용도로만 운영됩니다.
